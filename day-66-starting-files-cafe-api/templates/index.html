Cafe API Documentation

Introduction

This API enables CRUD operations on a cafe database.

Base URL

                http://localhost:5000/

Endpoints

GET /random

Returns a random cafe from the database.

Example Response:
                        {
                            "id": 1,
                            "name": "Cafe XYZ",
                            "location": "Buenos Aires",
                            ...
                        }
GET /all

Returns all cafes from the database.

Example Response:
                        [
                            {
                                "id": 1,
                                "name": "Cafe A",
                                ...
                            },
                            {
                                "id": 2,
                                "name": "Cafe B",
                                ...
                            }
                        ]
GET /search?loc=[location]

Returns cafes by a specific location.

Example Response:
                        [
                            {
                                "id": 1,
                                "name": "Cafe A",
                                "location": "Buenos Aires",
                                ...
                            }
                        ]
POST /add

Adds a new cafe to the database.

Example Request Payload:
                        {
                            "name": "Cafe C",
                            "location": "Mendoza",
                            ...
                        }
Example Response:
                        {
                            "success": "Successfully added the new cafe."
                        }
PATCH /update-price/[cafe_id]?new_price=[new_price]

Updates the coffee price of a specific cafe.

Example Response:
                        {
                            "success": "Successfully updated the price."
                        }
DELETE /report-closed/[cafe_id]?api_key=[api_key]

Deletes a cafe based on ID and a valid API key.

Example Response:
                        {
                            "success": "Successfully deleted the cafe."
                        }
